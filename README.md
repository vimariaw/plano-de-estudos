# 📚 Plano de Estudos

App de planejamento de estudos com calendário diário, semanal e mensal.

---

## Como colocar no ar (sem precisar saber programar)

### Passo 1 — Criar conta no GitHub (gratuito)
1. Acesse https://github.com e clique em **Sign up**
2. Crie sua conta com e-mail e senha

### Passo 2 — Criar um repositório
1. Clique no **+** no canto superior direito → **New repository**
2. Nome: `plano-de-estudos`
3. Deixe como **Public**
4. Clique em **Create repository**

### Passo 3 — Subir os arquivos
1. Na página do repositório, clique em **uploading an existing file**
2. Arraste **todos** os arquivos e pastas desta pasta para a área de upload
3. Clique em **Commit changes**

### Passo 4 — Publicar no Vercel (gratuito)
1. Acesse https://vercel.com e clique em **Sign up with GitHub**
2. Clique em **Add New Project**
3. Selecione o repositório `plano-de-estudos`
4. Clique em **Deploy**
5. Em menos de 1 minuto, você recebe um link tipo:
   `https://plano-de-estudos-seu-nome.vercel.app`

### Passo 5 — Adicionar à tela inicial
**iPhone/iPad:**
- Abra o link no Safari
- Toque no botão compartilhar (□↑)
- Toque em **Adicionar à Tela de Início**

**Android/tablet:**
- Abra no Chrome
- Toque nos 3 pontos (⋮)
- Toque em **Adicionar à tela inicial**

**Notebook:**
- É só salvar o link nos favoritos
